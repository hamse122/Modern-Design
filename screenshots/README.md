# Screenshots Guide

This directory contains screenshots of the Modern Design Portfolio 2025.

## How to Take Screenshots

### Option 1: Browser Developer Tools
1. Open your portfolio in Chrome at `http://localhost:8080`
2. Press `F12` to open Developer Tools
3. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
4. Type "screenshot" and select "Capture screenshot"
5. Save as `dark-mode.png` or `light-mode.png`

### Option 2: Browser Extension
1. Install a screenshot extension (e.g., "Awesome Screenshot")
2. Take full-page screenshots of your portfolio
3. Save them in this directory

### Option 3: Windows Snipping Tool
1. Open your portfolio in the browser
2. Press `Windows + Shift + S` for Snipping Tool
3. Select the area to capture
4. Save as PNG in this directory

### Recommended Screenshots:
- **dark-mode.png** - Full page screenshot in dark mode
- **light-mode.png** - Full page screenshot in light mode  
- **mobile-view.png** - Screenshot in mobile/responsive view (use DevTools device emulation)

## File Naming Convention
- Use descriptive names: `dark-mode.png`, `light-mode.png`, `mobile-view.png`
- Use PNG format for best quality
- Recommended size: 1920x1080 or larger for desktop, 375x812 for mobile
